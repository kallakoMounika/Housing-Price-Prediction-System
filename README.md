#  Housing Price Prediction System
 Project Overview
This project is a Machine Learning-based system that predicts housing prices using various features such as location, size, number of rooms, and amenities.

The model is built using Python and trained on housing datasets to provide accurate and data-driven predictions.

---

## Objectives
- Build a complete ML pipeline for price prediction
- Understand regression models and feature importance
- Achieve high accuracy with low error rates
- Perform data preprocessing, training, and evaluation

---

##  Problem Statement
Estimating housing prices manually is difficult due to multiple influencing factors.  
This project solves that problem using machine learning to provide reliable predictions.

---

##  Technologies Used
- **Language:** Python  
- **Platform:** Google Colab  
- **Libraries:**
  - pandas
  - numpy
  - matplotlib
  - seaborn
  - scikit-learn  

---

##  Features
- Data preprocessing (handling missing values, encoding, scaling)
- Exploratory Data Analysis (EDA)
- Model training using:
  - Linear Regression
  - Random Forest
- Model evaluation using:
  - MAE (Mean Absolute Error)
  - RMSE (Root Mean Squared Error)
  - R² Score
- Housing price prediction
